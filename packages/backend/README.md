# Backend Service

A Cloudflare Worker built with [Hono](https://hono.dev/).

## structure

- `src/index.ts`: Main entry point and route definitions.
- `wrangler.jsonc`: Cloudflare Workers configuration.

## Development

Run `npm run dev` from the root or `npm run dev` in this directory to start the local development server.

## Deployment

Deploy to Cloudflare Workers using `npm run deploy`.
